# ロリポップ 固定IPアクセス（WireGuard VPN）解約ガイド

## 経緯

DIPS 2.0 API連携の固定IP取得手段としてロリポップの固定IPアクセス（WireGuard VPN）を検討したが、さくらVPS（133.125.37.102）へ移行済みのため不要となった。

---

## 1. 解約前の確認事項

- **ロリポップのレンタルサーバー契約は別サービス**であり、VPNオプションのみの解約でサーバー契約には影響しない。レンタルサーバーを引き続き利用する場合はそのまま継続可能。
- 現在VPN経由でアクセスしているサービスや自動化スクリプトがないか確認する。
  - さくらVPS側でDIPS API Proxyが稼働中であれば、VPN経由の通信は既に使っていないはず。
- VPNに紐づく固定IPアドレスをDIPS 2.0の申請書類やホワイトリストに登録している場合は、さくらVPSのIPに変更済みか確認する。
- WireGuard設定ファイル（`.conf`）がローカルPCに残っている場合、解約後は不要になるので控えておく（念のため）。

## 2. 解約手順

1. ロリポップの**ユーザー専用ページ**にログインする。
2. メニューから**「契約・お支払い」**（または「オプション管理」等の該当メニュー）を開く。
3. 「固定IPアクセス」または「WireGuard VPN」のオプション項目を探す。
4. 該当オプションの**「解約」ボタン**をクリックし、確認画面で内容を確認して解約を確定する。
5. 解約完了のメールが届くことを確認する。

> 管理画面の構成は変更される場合があるため、該当メニューが見つからない場合はロリポップのサポートに問い合わせること。

## 3. 解約後の確認事項

- **ムームードメインのDNS設定には影響なし**。`dronepeak-dips.minato-morioka.jp` はさくらVPS（133.125.37.102）を向いており、ロリポップVPNとは無関係。
- ローカルPCのWireGuardクライアントからVPN設定（トンネル）を削除する。
- 不要になったWireGuard設定ファイル（`.conf`）をローカルから削除する。
- 解約後、ロリポップのレンタルサーバー側が正常に動作しているか念のため確認する（VPNオプション解約がサーバーに影響しないことの最終確認）。

## 4. 注意事項

- VPNオプションの解約は**契約期間の途中でも可能かどうか**、ロリポップの規約を確認すること。日割り返金の有無もあわせて確認。
- 解約後にVPNの固定IPは再利用できなくなる。万が一再度必要になった場合は新規契約となり、別のIPが割り当てられる可能性がある。
- ロリポップのレンタルサーバー自体を今後も使う予定がない場合は、サーバー契約の解約も別途検討する。
